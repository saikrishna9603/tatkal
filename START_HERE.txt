┌────────────────────────────────────────────────────────────────────────────────┐
│ 🆘 TROUBLESHOOTING - QUICK FIXES                                              │
└────────────────────────────────────────────────────────────────────────────────┘

PROBLEM: "Python not found"
FIX: Install Python 3.11+ from https://www.python.org/downloads/

PROBLEM: "Node not found"  
FIX: Install Node.js 18+ from https://nodejs.org/

PROBLEM: "Port 8000 already in use"
FIX (Windows): 
  netstat -ano | findstr :8000
  taskkill /PID <number> /F

PROBLEM: "Port 3000 already in use"
FIX (Windows):
  netstat -ano | findstr :3000
  taskkill /PID <number> /F

PROBLEM: "Module not found" (Backend)
FIX:
  cd backend
  pip install -r requirements.txt --upgrade

PROBLEM: "npm not found"
FIX: Restart terminal after installing Node.js

PROBLEM: Frontend won't compile
FIX:
  cd frontend
  rm -rf node_modules package-lock.json
  npm install
  npm run dev

PROBLEM: CORS errors in browser console
FIX: Make sure:
  - Backend running on localhost:8000 ✓
  - Frontend running on localhost:3000 ✓
  - Both healthy (no errors in terminal)

For more help, see: COMPLETE_STARTUP_GUIDE.md
